## Installation

1. Clone repository
    ``` shell
    git clone git@github.com:Dar3cz3Q-University/FoAI-Backend.git
    ```
2. Install dependencies
    ``` shell
    poetry install
    ```
3. Copy and fill environment file
    ``` shell
    cp .env.dist .env
    ```
4. Run application
    ``` shell
    poetry run dev
    ```
